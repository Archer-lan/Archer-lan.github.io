2022-04-18
项目
Lanlan

商品列表

表名：commodity

商品ID 商品图片(唯一 不为空) 名称 描述 价格 类别（不为空） 库存 状态 商店ID（不为空）
com_id image_id com_name com_descrip com_price type_id com_num com_status store_id
int int varchar(255) varchar(255) varchar(255) int int int int

主键：商品ID com_id

外键：商品图片 image_id

​ 类别 type_id

​ 商店ID store_id

商品状态：分为三种（0：在售，1：下架，2：删除）

图片表

表名：image

图片ID 图片地址 图片类型 图片描述
image_id image_url image_type image_descrip
int varchar(255) varchar(255) varchar(255)

主键：图片ID image_id

图片类型：按使用类型分类

类型表

表名：type

类型ID 类型名称
type_id type_name
int varchar(255)

主键：类型ID type_id

类型名称：（此分类不分二级目录，只分一级分类）

账号商店表

表名：store

商店ID 商店名称（唯一） 商店头像（唯一） 账号（唯一 不为空） 手机号（唯一 不为空） 密码（不为空） 商店描述
store_id store_name image_id store_account store_phone store_pwd store_descrip
int varchar(255) int varchar(45) varchar(11) varchar(45) varchar(255)

主键：商店ID store_id

外键：商店头像 image_id

其中：账号、手机号、密码不为空

用户信息表

表名：userinfo

用户ID 用户姓名（唯一） 用户手机（唯一） 用户地址
user_id user_name user_phone user_address
int varchar(45) varchar(11) varchar(255)

主键：用户ID user_id

物流信息表

表名：logistics

物流ID 用户ID（不为空） 发货日期 到货日期 物流状态 商店ID（不为空） 物流标签 物流号（不为空） 订单ID（不为空）
logi_id user_id logi_makedate logi_compdate logi_status store_id logi_flag logi_numb orders_id
int int varchar(45) varchar(45) int int int varchar(45) int

主键：物流ID logi_id

外键：用户ID user_id

​ 商店ID store_id

​ 订单ID orders_id

物流状态：0：未签收，1：已签收

物流标签：0：未删除，1：已删除

订单信息表

表名：orders

订单ID 订单编号（不为空） 用户ID（不为空） 商品ID（不为空） 订单支付金额 订单状态 商店ID（不为空）
orders_id orders_number user_id com_id orders_pay orders_status store_id
int varchar(45) int int varchar(45) int int

主键：订单ID orders_id

外键：用户ID user_id

​ 商品ID com_id

​ 商店ID store_id

订单状态：0：待支付，1：已支付
